# moss
### A helper app for [Hello IT](https://github.com/ygini/Hello-IT) to make building mobileconfig files easy!
```
               ___
              |   \__
              |      \_
        ______|        \__
       _|                 |
       \    -----         |    _----------------_
        \   |    \___   _/    |    Hello IT?     |
         ---|________|        |  Have you tried  |
           |[__]-[__]|        |   initiating an  |
            | ______ |        |    unexpected    |
            | |____| |        <      reboot?     |
             \      /          -________________-
              \____/
        _|----|{_}|----|_
        |      [ ]       |
        |      [ ]       |
```
